Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. Think of it as the skeleton that gives every website its basic structure.

Key Points About HTML:

  • HTML stands for HyperText Markup Language
  • HTML describes the structure of web pages
  • HTML consists of elements that tell the browser how to display content
  • HTML is not a programming language; it's a markup language
  • HTML is the standard markup language for creating Web pages

A Simple HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

Let's Break Down This Code:

  • <!DOCTYPE html> - Declares that this is an HTML5 document
  • <html> - The root element of the HTML page
  • <head> - Contains metadata about the document
  • <title> - Specifies a title for the page (shown in browser tab)
  • <body> - Contains the visible page content
  • <h1> - Defines a large heading
  • <p> - Defines a paragraph

Brief History of HTML

1991

Tim Berners-Lee created HTML as a way to structure documents for the World Wide Web

1995

HTML 2.0 was developed, adding forms and tables

1997-1999

HTML 3.2 and 4.0 introduced more features and standardization

2014

HTML5 became the official standard, bringing powerful new features
