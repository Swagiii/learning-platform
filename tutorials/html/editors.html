Creating Your First HTML Page

Step 1: Open Your Editor

Windows (Notepad):

  • Press Start
  • Type "Notepad"
  • Click on Notepad app

Mac (TextEdit):

  • Open Applications
  • Find TextEdit
  • Set to Plain Text mode (Format ‚Üí Make Plain Text)

Step 2: Write Your HTML

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Step 3: Save Your File

Important Tips:

  • Save with .html extension (e.g., "index.html")
  • Use UTF-8 encoding
  • Remember the file location

Step 4: View in Browser

Double-click your saved HTML file to open it in your default web browser.
