HTML Basics

Building Your First Web Page

Let's explore the fundamental building blocks of every HTML document and learn how to create your first web page.

HTML Document Structure

<!DOCTYPE html>
<html>
<head>
    <title>Basic HTML Structure</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a basic HTML document.</p>
</body>
</html>

DOCTYPE Declaration

Every HTML document must start with <!DOCTYPE html> to tell browsers it's an HTML5 document.

HTML Element

The <html> element is the root element containing all other elements.

Head Section

The <head> contains metadata about the document.

Body Section

The <body> contains all visible content.

Essential HTML Elements

Headings

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Smaller Heading</h3>
<h4>Even Smaller</h4>
<h5>Small Heading</h5>
<h6>Smallest Heading</h6>

Paragraphs

<p>This is a paragraph of text.</p>
<p>This is another paragraph.</p>

Links

<a href="https://example.com">Click here to visit</a>

Images

<img src="image.jpg" alt="Description of image" width="300" height="200">
